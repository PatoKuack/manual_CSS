Vinculación entre lenguajes

Para vincular un archivo con terminación HTML y CSS se usa la etiqueta link sin finalización (self closing) en la sección HEAD del documento HTML.
Con su atributo rel="stylesheet" se indica que es un archivo de estilo, el que se vinculará.
Con el atributo href="ruta_del_archivo.css" se indica la ruta del archivo .css que se vinculará.
Si el archivo CSS está en una carpeta exterior a la del HTML, se escriben dos veces un punto y un slash o diagonal para salirse un nivel de la carpeta y explorar desde ahí.
E:
˫---Pages
|   Page1.html
|   Page2.html
|
˫---Styles
|   Style1.css
|   Style2.css
|
˪---Imges
    Image1.css
 Aquí el archivo "Prueba1" se vincularía con "style1" pero están en carpetas diferentes, así que se vincularía así: href="../Estilos/style1.css"
Esta es la forma más recomendada de vincular archivos HTML y CSS.

También se puede colocar desde el archivo HTML un estilo con la etiqueta style con finalización en la sección HEAD.
Dentro de esta etiqueta se coloca el código que iría en el archivo CSS.
En esta forma el estilo CSS se aplica solo a este documento y la idea es que sea a toda la página y sus vínculos.
O bien, en vez del código se coloca @import url(ruta_del_archivo.css) para vincular los archivos HTML y CSS.
Con esta forma del @, se tiene el inconveniente de que si falla la carga en el archivo importado afectará la página.

También se puede colocar el estilo desde las etiquetas, por ejemplo, en la etiqueta "h1" se puede colocar como atributo style="color: purple"
Esta forma de aplicar CSS solo se aplica a una parte en específico de la página.

Sintaxis en CSS

h1{
 color:   green;
 background: orange;
}
Los comentarios en CSS se colocan con los símbolos slash y asterisco /* */
