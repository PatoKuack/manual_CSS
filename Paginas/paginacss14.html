Holy grail layout.

Es un layout que tiene tres renglones, uno para el encabezado y menú de navegación (va al inicio), uno para el pie de página (va al final) y uno para el contenido principal y publicidad (va en medio), se debe colocar el contenido de la página (contenido principal y publicidad) en columnas, las cuales deben tener la misma proporción de altura.

Columnas.

Los layouts, sobre todo los de tipo holy grail, se hacen con grid o flexbox, sin embargo existe un conjunto de propiedades que ayudan a manipular el contenido de los elementos separandolos por columnas, haciendo un buen complemento para crear layots más eficientemente:

column-count: #;
Se coloca en el elemento contenedor. Requiere un valor numerico para indicarle cuantas columnas debe cubrir, a esas columnas les distribuye (no separa) el contenido.
column-width: #[px];
Se coloca en el elemento contenedor. Es una propiedad complemento, esta propiedad requiere de un valor de medición con el que le indicará a las columnas cuanto deben medir de ancho cada una si es que quieren dividirse, es decir, si las columnas no tienen espacio suficiente en el contenedor para medir lo indicado en esta propiedad, mantendrán el contenido en un minimo de columnas.
columns: #[em] #
Se coloca en el elemento contenedor. Esta propiedad es un shorthand se las dos primeras propiedades, recibe el ancho minimo de las columnas para dividirse y el numero de columnas.
column-gap: #[rem]
Se coloca en el elemento contenedor. Esta propiedad recibe un valor de medición para indicar la separación entre las columnas.
column-rule: #[px] line-type color;
Se coloca en el elemento contenedor. Funciona como el shorthand de la propiedad border, coloca un borde entre las columnas. Si las columnas tienen separación se coloca el borde justo en medio de la separación de las columnas.
column-span: [ none | all ];
Se coloca en un elemento (como hr o aside). Separa el contenido de todas las columnas de una forma horizontal.
column-fill: [ balance | auto | ];
Se coloca en el elemento contenedor. Tiene 2 valores:
  • balance: distribuye el contenido de una forma balanceada
  • auto: rellena una columna con el contenido antes de rellenar la siguiente, este valor requiere que el contenedor tenga una altura fija.
