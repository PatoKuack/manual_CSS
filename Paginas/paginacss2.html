Custom properties (variables CSS)

Es la reutilización de valores en el código.

Las variables en CSS se declaran dentro de un selector, su nombre siempre comienza por dos guiones altos (--), distingue entre mayúsculas y minúsculas (case sensitive) y se mandan a llamar con el función var() colocándole como parámetro la función que se desea utilizar.

:root {
 --color: purple;
}
h1 {
 color:  var(--color);
}
p {
 color:  var(--color);
}
Es recomendable colocar nuestros estilos repetitivos en nuestra clase :root porque esta raíz es accesible a todo nuestro código.

Selectores

Indican a que elemento de HTML se le aplican los estilos.

Selectores simples

Son de una sola palabra (ejemplo: h1{})
Si puedes, no los utilices en CSS.

Etiquetas, tipos o elementos

Sirven para normalizar el estilo, que el estilo en la página sea igual en todos los navegadores.
y en toda la página, por ejemplo, si aplico un color a los párrafos, todos los párrafos serán de ese color.
h3 {
 color: brown;
}

Clases

Son los más utilizados en CSS (prácticamente cada etiqueta debería tener una clase) y permiten definir a que darle un estilo y a que no dárselo mediante clases,
son reutilizables y se pueden declarar en HTML más de una clase separándolas en el atributo class con un espacio
(ejemplo: class="greenone redtwo purplethree").
Tienen unas reglas en su declaración en HTML:
solo pueden comenzar con letra, guion bajo o guion alto (nunca con números u otros caracteres).
Su nombre pueden ser de una sola letra.
HTML:
<h4 class="titlegreen">Clases</h4>

CSS:
.titlegreen {
 color: green;
}
Se indican en CSS con: un punto y el nombre de la clase.

Id

Utilizan los identificadores de HTML, no hay dos identificadores iguales, no son reutilizables (por lo que es preferible reservarlos para JavaScript) y es difícil sobrescribirlos en cascada como los demás.
Los identificadores se usan para anclajes en la página HTML o para seleccionar elementos, pero casi no se usan en CSS.
HTML:
<hr id="linea">

CSS:
#linea {
 color: cyan;
}
Se indican en CSS con: un numeral (#) y el nombre del id.

Selectores compuestos

Son los que llevan más de una palabra.

Agrupados (usando comas)

Se pueden agrupar selectores al colocarle un estilo a más de un selector.
HTML:
<h4 class="_Agrutitle">Agrupados</h4>
<h4 class="_Combititle">Combinados</h4>
<h4 class="_Descentitle">Descendientes</h4>

CSS:
._Agrutitle, ._Combititle, ._Descentitle {
 background: bisque;
}

Combinados (sin separaciones)

Se colocan más de un selector (generalmente 2), le pide al navegador que aplique el estilo en los elementos que tengan los selectores pedidos.
(ejemplo: h3.classtitle#idtitle)
h4._Combititle {
 color: fuchsia;
}
No deben tener espacios entre selectores en la declaración en CSS.

Descendientes (usando espacios)

Se colocan los selectores de modo descendiente, es decir, si body contiene a la etiqueta p (aunque este no se coloca),
y p contiene a div, y div a spam, en este caso se colocarían los selectores así:
body div spam{}
Se utilizan generalmente en items de menús para aplicar estilos a descendientes de ciertos elementos.
Deben tener espacios entre selectores en la declaración en CSS.
body div {
 background: powderblue;
}

Selectores con operadores

Contienen algún operador (ejemplo: h1 + span{}).
Nos ayudan a crear selectores compuestos con una relación más especifica
    * selector universal: selecciona todo.
    > hijo directo: selecciona una relación especifica de hijo directo, ejemplo:
        dentro de div estaría spam (spam es hijo directo de div) pero spam no es hijo directo de body (porque spam esta 2 niveles abajo de body).
        la etiqueta p no se reconoce dentro de la descendencia.
    + hermano siguiente: selecciona todo elemento definido del mismo nivel de descendencia (hermanos) que esté después del primer elemento especificado, ejemplo:
        b + u aplicará el estilo a todo elemento u que esté después de b,
        si se tuvieran etiquetas u antes de b y dentro del mismo nivel de descendencia, no se le aplicaría el estilo.
    ~ hermanos siguientes: selecciona, al igual que el operador +, los elementos hermanos definidos que estén después del primer elemento definido,
        pero esta vez aplica el estilo a todos los siguientes, no solo al siguiente.
* {
 font-family: 'Courier New', Courier, monospace;
}
pre > b {
 color: blueviolet;
}
b + u {
 background-color: burlywood;
}
i ~ u {
 background-color: yellow;
}

Selectores de atributo

Usan cualquier atributo de HTML, usan corchetes para su declaración en CSS, ejemplo:
[title]{} aplica el estilo a cualquier elemento de la página que tenga el atributo title.
[width="300"]{} aplica el estilo a cualquier elemento de la página que tenga el atributo width y que su valor sea de 300.
Hay selectores de atributos para no indicar exactamente el valor del atributo usando los símbolos
cater (^) para "todo aquello que inicie con",
pesos ($) para "todo aquello que termine con" y
asterisco (*) para "todo aquello que contenga", ejemplo:
[href^="http://"]{} aplicará un estilo a todo aquello que en su atributo "href" tenga un valor que inicie con el protocolo no seguro (http)
[href$=".mp4"]{} aplicará un estilo a todo aquello que en su atributo "href" tenga un valor que termine con el formato de almacenamiento .mp4
[href*="youtube"]{} aplicará un estilo a todo aquello que en su atributo "href" contenga entre su valor la palabra "youtube".
Estos tres últimos tres selectores de atributos sirven para identificar rutas, elementos o patrones.

[width^="3"] {
 border-radius: .5cm;
}
