Canal alfa.

Dependiendo del modo de color, una imagen tiene uno o varios canales que definen cómo son los colores.

Un canal alfa es un canal adicional que define la transparencia que tiene cada pixel de la imagen.

Algunos formatos de imagen admiten canales alfa, como PNG o TIFF, pero otros como JPEG no.

Máscaras.

Entre shape y clip-path la diferencia es el comportamiento mientras que entre clip-path y mask es el área delimitada que se mostrara en la imagen, o sea que con la propiedad mask se usa el canal alfa de un elemento (en la capa superior) para hacer el recorte de otro elemento (en la capa inferior).

Esta función esta parcialmente soportada por los navegadores por lo que hay que utilizar -webkit- antes de cada propiedad, solamente Firefox lo soporta totalmente en la actualidad (diciembre del 2021).

Esta propiedad tiene un comportamiento similar a cuando se usa un fondo como background-repeat, size, position, etc.

mask-image: url('___');
Recibe la URL de la imagen.
mask-repeat: [repeat | no-repeat | repeat-x | repeat-y];
Por default se repite el recorte, pero se puede ordenar que no se repita o que se repita en el eje X o en el eje Y.
mask-position: [top | right | bottom | left | center];
Por default se coloca el recorte en la parte superior izquierda (en perspectiva del usuario), pero se puede elegir la posición deseada.
mask-size: [auto | contain | cover | #];
Define el tamaño del recorte, por defecto tiene el tamaño del elemento original.
mask: _ _ _ / _;
Se puede utilizar el shorthand de las 4 propiedades anteriores.
.img {
 mask: url('/logo.svg') no-repeat center / 200px;
}
mask-image: linear-gradient()
Esta propiedad no trabaja con el canal alfa de otro elemento, simplemente crea una capa encima de la imagen y genera el degradado lineal.
mask-image: radial-gradient()
Esta propiedad no trabaja con el canal alfa de otro elemento, simplemente crea una capa encima de la imagen y genera el degradado radial.
mask-composite: [add | subtract | intersect | exclude]

Se utiliza con dos o más elementos que recortan a una imagen, es decir, dos o más máscaras (ver el ejemplo de código).

  • add: Combina el canal alfa de las máscaras haciéndolas una sola.
  • subtract: Cuando dos máscaras tienen intersecciones de área, la máscara frontal es eliminada y el área interceptada es eliminada de la máscara trasera.
  • intersect: Cuando dos máscaras tienen intersecciones de área, las máscaras son eliminadas dejando únicamente el área interceptada.
  • exclude: Cuando dos máscaras tienen intersecciones de área, la intersección de las máscaras es eliminada dejando el resto de las máscaras.
.div {
 // Se pueden agregar dos o más máscaras diferentes o la misma mascara dos veces
 mask: url('/circle.png'), url('https://...');
 mask-size: 100% 3rem;
 mask-position: center, 100px center;
 mask-repeat: no-repeat;
 mask-composite: subtract;
}
