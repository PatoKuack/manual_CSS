Librerías.

Se colocan abajo de las etiquetas meta y se aplican con las etiquetas link, se colocan los links por prioridad.
Por ejemplo con la página Normalize css se puede adquirir una normalización de tu documento HTML para que en cualquier navegador no se le apliquen los atributos CSS que están por default en algunas etiquetas como los márgenes de los títulos o los colores de los enlaces.

Iconos

Mediante la página Jam Icons se puede elegir y colocar un icono mediante CSS.
Primero nos dirigimos al enlace Use:
Luego se busca la sección de CSS/Font y en la segunda opción se muestra el código para incorporar la librería a nuestro código, se copia el código y se coloca en el documento HTML, debajo de los meta en la sección head.
Después para poder usar algún icono se debe colocar un span y dentro de la etiqueta se coloca la clase como se indica en la primera opción de la sección CSS/Font de la página JAM Icons, se coloca span class="jam jam-nombre del icono":
Ejemplo del icono "Whatsapp":

Para agregar un icono a la pestaña de la página se escribe el comando
<link rel="shortcut icon" type="image/x-icon" href="ruta_del_archivo.ico">,
rel indica que será un icono y type indica el tipo de imagen que será.
En la página online-convert se pueden transformar imágenes pequeñas en .ico.

Nota:
De preferencia siempre se trata de colocar agradecimientos a los autores de los iconos, imágenes o documentos en la página en donde se ocupan.
