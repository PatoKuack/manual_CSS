Posicionamiento de elementos.

El posicionamiento es la forma de poder ubicar un elemento en cualquier parte del layout. Para manipular el posicionamiento se utiliza la propiedad position.

Esta propiedad recibe 5 valores:

position: [ static | relative | absolute | fixed | sticky ]

Static es el valor por defecto que tienen todos los elementos e indica que un elemento no está posicionado.

Propiedades offset.

Algo que ayuda a manejar el comportamiento de los posicionamientos (no incluido a static) es las propiedades offset:

  • top
  • right
  • left
  • bottom

Estas propiedades permiten posicionar un elemento en diferentes lugares de la ventana y solo están disponibles para elementos posicionados.

Con estas propiedades se dice a partir de donde empezara a moverse, es decir, si le asignamos top: 20px; a un elemento X, el elemento X se desplazará 20 pixeles a partir de la parte superior de la ventana del navegador hacia la parte inferior (se podría decir que se desplaza hacia la parte contraria a la especificada top->bottom).

Las propiedades buscan la referencia del elemento y se mueve con respecto a ella. La referencia de cada elemento depende del posicionamiento que obtienen con la propiedad position.

Se pueden asignar valores negativos en estas propiedades y cualquier unidad de medida, solo que para trabajar con porcentajes en forma vertical se tiene que asignar una altura fija en el elemento padre del elemento, ya que la propiedad toma la altura del elemento padre para sacar el porcentaje de desplazamiento y sin una altura fija el porcentaje que obtiene es 0.

Relative.

Este valor (relative) no provoca que el elemento pierda su espacio reservado dentro de la estructura de la página.

La posición inicial y la referencia (con respecto a las propiedades offset) de un elemento, cuando se utiliza relative, es la misma que se le asigna al elemento originalmente dentro de la página.

Con ayuda de las propiedades offset se pueden desfasar los elementos de posicionamiento relativo de su espacio reservado sin que otros elementos invadan este espacio reservado o sin que el elemento con posicionamiento relativo desface a otros elementos evitando que modifiquen su posición en la página.

Absolute.

Este valor (absolute) provoca que el elemento pierda su espacio reservado dentro de la estructura de la página recorriendo los elementos que le suceden tratando al elemento absoluto como si fuera una imagen de fondo.

La posición inicial y la referencia (con respecto a las propiedades offset) de un elemento, cuando utiliza absolute, es el ancestro (contenedor padre) más cercano que este posicionado (con relative, fixed o sticky).

Debido a este tipo de referencia, se le puede asignar dos veces una propiedad offset a un espacio, por ejemplo, un left: 5px; a un contenedor relative y otro left: 5px; a su contenido absolute.

El elemento con posición absoluta limita su espacio al contenido que tiene.

Las unidades de medida del elemento absoluto dependen de su contenedor padre, por ejemplo, los porcentajes toman de referencia el tamaño del contenedor padre.

Fixed.

Este valor (fixed) provoca que el elemento pierda su espacio reservado dentro de la estructura de la página recorriendo los elementos que le suceden tratando al elemento absoluto como si fuera una imagen de fondo.

La posición inicial y la referencia (con respecto a las propiedades offset) de un elemento, cuando utiliza fixed, es el viewport (ventana del navegador).

El elemento con posición fija (fixed) limita su espacio al contenido que tiene.

Cuando se posiciona un elemento con fixed, el elemento queda fijo y no importa en que parte de la página esta, se moverá a la par del scroll.

Sticky.

Este valor (sticky) no provoca que el elemento pierda su espacio reservado dentro de la estructura de la página.

La posición inicial y la referencia (con respecto a las propiedades offset) de un elemento, cuando se utiliza sticky, puede ser el scroll o la parte superior de la página.

Ya que su referencia es la parte superior del viewport, entonces solo funciona con la propiedad offset top y actúa de una forma particular, es decir:

Si a un elemento pegajoso (sticky) se le coloca un top: 40px;, el elemento se desplazará junto al scroll tal como lo hace fixed, pero lo hará sólo cuando la barra de scroll baje 40px. Así que el top en este caso funcionaría como una condicional.
