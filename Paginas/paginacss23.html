Pseudo-elementos.

Pseudo se refiere a cualquier cosa que aparenta ser lo que no es, por lo que un pseudo-elemento es una propiedad que aparenta ser un elemento pero no lo es.

Los pseudo-elementos permiten crear selectores más avanzados sin necesidad de agregar mucho código al programa. Hay dos tipos de pseudo-elementos.

  • Generador de contenido: pretenden ser un elemento (elemento = cualquier etiqueta en el archivo).
  • Seleccionador de contenido: manipulan una parte especifica del contenido de un elemento.

Los pseudo-elementos tienen 3 especificaciones puntuales para su correcta declaración, uso y funcionamiento:

  • Requieren el uso de 4 puntos para funcionar (::).
  • Necesitan ser usados en elementos que puedan encapsular a otros (.h2::before).
  • No se puede acceder a un pseudo-elemento en JavaScript porque no existe en el DOM, no se pueden seleccionar en la página y tampoco se puede ver su valor en el inspector de elementos del navegador.

Generadores de contenido.

Los principales pseudo-elementos generadores de contenido son ::after y ::before

La forma en que funcionan los pseudo-elementos es la siguiente:

::before

Ejecuta los estilos de su bloque al inicio del selector especificado. En el siguiente ejemplo se insertará un texto con la propiedad content antes del título h2:

h2::before {
 content: "Bubalo-";
}
// El título quedaría: Bubalo-TítuloH2
::after

Ejecuta los estilos de su bloque al final del selector especificado. En el siguiente ejemplo se insertará un texto con la propiedad content después del título h2:

h2::after {
 content: "-bubalo";
}
// El título quedaría: TítuloH2-bubalo

Todo pseudo-elemento vacío (con un content: '';) debe ser forzosamente un bloque (display: block;). Se utilizan los pseudo-elementos vacios para colocar imágenes como fondo (background: URL.svg;) y redimencionarlas (background-size: contain;) o para dibujar.

.container::before, .container::after {
 content: '';
 display: block;
}

El objetivo del dibujo es hacer algo que no se pensaría que está hecho con código. Para hacer un dibujo eficiente se debe utilizar la menor cantidad de elementos posibles.

Una función que es de mucha ayuda para los pseudo-elementos es attr(). atrib() obtiene su contenido del atributo que se encuentra en su parámetro, es decir, obtiene su contenido desde la estructura HTML.

Un atributo que se utiliza con la función atrib() es data="". data es un tipo de atributo personalizado en el cual se acostumbra a guardar información.

<p data-money="$" data-code="MX" class="p">
 30
</p>
.p::before {
 content: 'Precio: ' attr(data-money;);
}
// Resultado en la página: Precio: $30

Seleccionadores de contenido.

Los principales pseudo-elementos que seleccionan contenido que ya esta en el DOM son ::first-line, ::first-letter y ::selection

::first-line

Sirve para seleccionar la primera línea de un parrafo, lista o cualquier contenido.

No importa si el texto se modifica al adaptarse a un ancho mayor o menos, ::first-line seguirá seleccionando solo la primera línea del texto.

::first-letter

Sirve para seleccionar la primera letra de un parrafo, lista o cualquier contenido.

Como se utiliza en textos, se suele utilizar en su bloque de estilos las propiedades float: left; para que la letra seleccionada pueda ocupar más de un renglón sin perder su espacio reservado y line-height: 1; para centrar la letra seleccionada entre los renglones que abarca.

::selection

Sirve para seleccionar la selección de un parrafo, lista o cualquier contenido, es decir, aplica estilos a un fragmento de texto sombreado, el cual es seleccionado por el usuario para copiar o cortar dicho fragmento de texto.

Este pseudo-elemento está limitado a modificar solamente: el color, el fondo y la sombra del texto.

Existen otros pseudo-elementos como ::backdrop, pero son nuevos y no todos los navegadores los soportan.
