Formato para fuentes digitales.

A inicios de la web cada navegador creaba sus propias fuentes y se decidió estandarizar creando 5 principales tipos de fuente digitales de los cuales WOFF o WOFF2 son los más recomendables para usarlos en un proyecto web.

Es importante tener una licencia para poder usar la fuente en un proyecto sin meterse en problemas legales.

La página más famosa para descargar formatos de fuentes es Font squirrel, con su opción "generator" se puede subir una fuente en cualquier formato y transformarlo en formato WOFF2 que es el más recomendado.

También hay una página en donde se pueden comprar formatos gratis es en My fonts.

Tipos de formatos digitales para fuentes.

True Type Font (TTF).
Se usa en cualquier sitio.
Open True Type Font (OTF).
Facilita o implementa la creación de las curvas Bézier utilizadas por los diseñadores para crear tipos de fuentes, pero en cuestión de uso es lo mismo que TTF.
Embedded Open Type (EOT).
El formato se maneja de forma comprimida por lo que pesa menos y resulta un buen recurso para las páginas web. También detecta las fuentes no licenciadas usadas de forma incorrecta por diseñadores web, sin embargo, este formato solo lo soporta Internet Explorer (se quedó como una propuesta).
Web Open Font Format (WOFF) (WOFF2)
Se maneja por metadatos lo cual lo vuelve más ligero que la compresión de EOT, mejorando el rendimiento de las páginas web.
Scalable Vector Graphics (SVG)
Se utilizan las fuentes como vectores, pero genera muchas desventajas como el de hacer una selección por letra y no por palabras o párrafos, otro problema es que no es muy legible.

@font-face

Cuando se importan fuentes de Google fonts se obtiene una URL en donde se encuentra un texto CSS con varias fuentes con una sola diferencia entre ellas, los rangos unicode que varían entre cada lenguaje, por lo que en cada fuente se anota con un comentario el tipo de lenguaje.

@import url('https://fonts.googleapis.com/css2?family=Redacted+Script:wght@300&display=swap');

.body {
 font-family: 'Redacted Script', cursive;
}

Cada fuente em el documento de la URL tiene una estructura con la que se obtienen las fuentes.

@font-face
Señala a un archivo de fuente digital específico para obtener datos.
font-family: '__';
Aquí se especifica el nombre de la fuente, este nombre se utiliza en otro elemento como un @import (en CSS) o en un <link> (en HTML) para poder utilizar la fuente en la página web.
font-style: __;
Indica el tipo de fuente, como normal, italic u oblique.
font-weight: __;
Indica el grosor de la fuente que por defecto son 400.
font-display: __;

Permite controlar el comportamiento de las fuentes al momento que se imprimen en pantalla, ayuda cuando se tiene una red lenta o rendimiento bajo. Tiene 5 valores:

  • Auto copia el comportamiento de block.
  • Block no muestra el contenido hasta que se descargue la fuente que se desea.
  • Swap primero muestra una fuente alternativa y una vez que obtiene la fuente la muestra.
  • Falllback por una fracción de segundo (1/10 segundo) desaparece el texto, si en ese lapso de tiempo no aparece la fuente que se desea, se muestra una alternativa y una vez que obtiene la fuente la muestra.
  • Optional por una fracción de segundo (1/10 segundo) desaparece el texto, si en ese lapso de tiempo no aparece la fuente que se desea, se analiza la conexión del usuario y si es lenta se asigna la fuente alternativa y ya no carga la fuente deseada.
src:

Señala al archivo que se quiere importar, para esta acción utiliza lo siguiente:

local('__'),
Se colocan los "local" que se consideren necesarios y se usa para que el archivo que utiliza la fuente busque las variaciones de nombre que tenga la fuente dentro del entorno local del proyecto y no tenga que descargar (o volver a descargar la fuente) o utilizar fuentes externas que consuman recursos del proyecto haciendo que la página web se cargue más lento en el navegador.
url('__')
Si en el entorno local no encuentra los nombres que se le indicaron, se redirecciona la búusqueda a la URL especificada para señalar la fuente que se desea importar, ya sea una URL de un archivo local o de un servidor.
format('__');
Indica el formato de la fuente digital.
unicode-range: _, _, _;
Especifica los caracteres especiales que se utilizarán en cada idioma en que se esté imprimiendo la fuente.

Para utilizar una fuente descargada con su licencia se utiliza el mismo código de la ley @font-face
