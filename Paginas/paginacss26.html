Tablas.

En CSS se le pueden dar a los elementos comportamientos de tablas con la propiedad display.

  • display: table; hace que el elemento se comporte como una etiqueta table de tipo bloque (con display block).
  • display: inline-table; hace que el elemento se comporte como una etiqueta table de tipo línea (con display inline).
  • display: table-caption; hace que el elemento se comporte como una etiqueta caption.
  • display: table-header-group; hace que el elemento se comporte como una etiqueta thead.
  • display: table-row-group; hace que el elemento se comporte como una etiqueta tbody.
  • display: table-footer-group; hace que el elemento se comporte como una etiqueta tfoot.
  • display: table-row; hace que el elemento se comporte como una etiqueta tr.
  • display: table-cell; hace que el elemento se comporte como una etiqueta td.
  • display: table-column; hace que el elemento se comporte como una etiqueta col.
  • display: table-column-group; hace que el elemento se comporte como una etiqueta colgroup.

A los captions se les puede asignar una posición superior (para titular la tabla) que es la de default o una inferior (para dar una descripción a la tabla) con la propiedad caption-side [ top | bottom ].

A las celdas se les puede asignar una propiedad para ocultarlas sin que pierdan su lugar dentro de la tabla, la propiedad es empty-cells: [ show | hide ]; tiene dos valores, con show muestro el contenido (es el valor por defecto de las celdas) y con hide se ocultan las celdas sin modificar la estructura de la tabla.

table :nth-of-type(2) {
 empty-cells: hide;
}

Las columnas siempre modifican su ancho independientemente de las demás, tomando como referencia la celda (de la columna) con el contenido más amplio, para que todas las columnas de la tabla tomen el mismo ancho tomando como referencia la celda (de toda la tabla) con el contenido más amplio, se utiliza la propiedad table-layout: [ auto | fixed ] ; la cual tiene dos valores, auto es el valor por defecto y fixed es el valor que obliga a todas las celdas a tener el mismo ancho.

table {
 table-layout: fixed;
 width: 100%;
}

Bordes.

Para eliminar el borde doble de las tablas se utiliza la propiedad border-collapse con la propiedad collapse que contrae los espacios entre los bordes. El valor por default que traen las tablas es separate.

 border-collapse: [ collapse | separate ];

También se le puede agregar una separación especifica a los bordes de las tablas con la propiedad border-spacing y se le agregan los valores de las separaciones entre las filas y un segundo valor para la separación entre columnas.

 border-spacing: (filas) (columnas);
