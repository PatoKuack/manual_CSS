Scroll.

El scroll es la barra de desplazamiento lateral o inferior que coloca el navegador para moverse entre los contenedores y visualizar todo su contenido.

Desbordamiento.

Para manipular la aparición del scroll dentro de un elemento con más contenido del que puede almacenar (contenido desbordado) se utiliza la propiedad overflow.

overflow tiene los siguientes valores:

  • visible valor por defecto, indica que el contenido que se desborda será visible.
  • hidden oculta el contenido desbordado.
  • scroll muestra las barras de scroll (la horizontal y la vertical) y las coloca, aunque el contenido no se esté desbordando.
  • auto coloca las barras de scroll cuando el contenido se desborda ya sea de manera horizontal, vertical o ambos.

Se puede manipular específicamente el eje X (horizontal) o el eje Y (vertical) con las propiedades overflow-x y overflow-y. Estas propiedades manejan los mismos valores que el overflow simple aplicando los valores a su eje, pero cuando se le coloca a un eje un tipo de scroll (por ejemplo: overflow-x: scroll;), el otro eje obtiene automáticamente un valor auto (por ejemplo: overflow-y: auto;) y esto afecta el contenido de los elementos a los que se le aplica esta propiedad.

Cuando se tienen elementos flotantes usando la propiedad float (sección 9: Recortes y figuras.) y se manipula el scroll, el contenido evita colocarse debajo o por encima del elemento flotante para evitar efectos negativos con la interacción del elemento flotante y el contenido al momento de desplazar el scroll.

Anclaje.

Un anclaje de enlaces es cuando se manda al usuario a una sección específica del contenido de un elemento.

Desplazamiento suavizado.

Con la propiedad y el valor scroll-behavior: smooth; se crea un desplazamiento suabe entre estos valores. Esta propiedad tiene tres desventajas:

  1. Solo funciona en elementos con scroll horizontal o vertical.
  2. No se puede controlar la transición del desplazamiento.
  3. No funciona si la propiedad se aplica en el body, en cambio funciona en cualquier otro elemento, hasta en el html.

Padding al scroll.

También se puede utilizar la propiedad scroll-padding con los anclajes para evitar que un menú que navegue por las secciones tape el contenido colocando una separación entre el scroll y el inicio de la sección a la que se redireccionó.

Esta propiedad se aplica al elemento padre (el menú que redirecciona a otros elementos) y como valor se le coloca una unidad de medida o 4 dependiendo:

  • scroll-padding-top: #;
  • scroll-padding-right: #;
  • scroll-padding-bottom: #;
  • scroll-padding-left: #;
  • scroll-padding: # # # #;

Margin al scroll.

La propiedad scroll-margin funciona igual que la propiedad anterior, excepto que se coloca en el elemento hijo (la sección a la que se redirecciona el menú), coloca un margin entre el scroll y el elemento anclado y se puede aplicar a las secciones en conjunto o individualmente.

  • scroll-margin-top: #;
  • scroll-margin-right: #;
  • scroll-margin-bottom: #;
  • scroll-margin-left: #;
  • scroll-margin: # # # #;

Ajustar desplazamiento.

La propiedad scroll-snap-type: [orientación alineación]; ayuda a ajustar el desplazamiento cuando se hace por medio de la rueda del mouse, se coloca en el elemento padre (en el menú) y lo que hace es que cuando detecta que el desplazamiento está por llegar a una sección, el scroll se ajusta para que la sección quede alineada en la ventana del navegador.

Esta propiedad necesita otra propiedad para funcionar, scroll-snap-align: [alineación]; se coloca en el elemento hijo (en las secciones) y lo que hace es indicar en que parte de la sección se ajustará el scroll.

scroll-snap-type tiene 5 tipos de valores:

Orientación.

  • x indica que la orientación del scroll es horizontal.
  • y indica que la orientación del scroll es vertical.
  • both indica que la orientación del scroll es horizontal y vertical.

Alineación.

  • proximity cuando el scroll detecta que la ventana del navegador está a punto de avandonar una sección, se desplaza automáticamente hacia el inicio, centro o final de la siguiente o anterior sección.
  • mandatory cuando el scroll detecta la aparición de una sección en la ventana del navegador, se desplaza automáticamente hacia el inicio, centro o final de la sección.

scroll-snap-align tiene 3 valores:

  • start indica que el scroll se alineará al inicio de la sección.
  • center indica que el scroll se alineará en medio de la sección.
  • end indica que el scroll se alineará al final de la sección.

scroll-snap-type se puede utilizar junto con scroll-padding o scroll-margin.

Ejemplo:

.html{
 scroll-snap-type: y mandatory;
 scroll-padding-top: 20vh;
 /* Se le aplica a la etiqueta html porque es el elemento que tiene scroll, pero se puede agregar un scroll a cualquier otro elemento y aplicarle estas propiedades a ese elemento */
}
.html-body-sections{
 scroll-snap-align: start;
}

Posiciones de ajuste.

La propiedad scroll-snap-stop: [normal | always]; en su valor always permite detener el scroll en una sección de anclaje para evitar que se salte rápidamente a la siguiente, es útil cuando se utiliza un ajuste de scroll de tipo mandatory. El valor por default es normal.

Actualmente (diciembre del 2021), Firefox es el único que no soporta esta propiedad.

Pseudo-elementos del scroll.

No todos los navegadores tienen soportados los pseudo-elementos de scroll. Estos pseudo-elementos se utilizan con -webkit-.

::-webkit-scrollbar

Permite cambiar dimensiones, color de fondo, sombras, borde... y hasta usar las pseudo-clases :active y :hover en el scroll de un elemento.

Cuando se le aplican estilos al scroll con esta pseudo-clase, desaparecen todos los elementos del scroll para que se comience a modificar uno por uno.

La aplicación de width es para modificar el ancho del scroll vertical y height para el horizontal.

::-webkit-scrollbar-track
Aplica estilos a todo lo que representa la barra del scroll.
::-webkit-scrollbar-track-piece
Aplica estilos a la parte del scroll que no está cubierta por la barra deslizable.
::-webkit-scrollbar-button
Aplica estilos a los botones del scroll.
::-webkit-scrollbar-thumb
Aplica estilos a la barra deslizable del scroll.
::-webkit-scrollbar-corner
Aplica estilos a la caja en la esquina que representa intersección de las barras, requiere que se le coloque un display: block; para funcionar.

Se le pueden aplicar pseudo-clases especiales a estos pseudo-elementos:

:vertical
Aplica el estilo del pseudo-elemento únicamente a la barra vertical.
.div::-webkit-scrollbar-thumb:vertical:hover {
 background-color: orange;
}
:horizontal
Aplica el estilo del pseudo-elemento únicamente a la barra horizontal.
:start
Solamente se puede utilizar con los pseudo-elementos button o track-piece, lo que hace es aplicar el estilo a los botones del inicio o a las partes del inicio del scroll que no cubre la barra deslizable (track-piece).
:end
Solamente se puede utilizar con los pseudo-elementos button o track-piece, lo que hace es aplicar el estilo a los botones del final o a las partes del final del scroll que no cubre la barra deslizable (track-piece).
:increment
Solamente se puede utilizar con los pseudo-elementos button o track-piece, lo que hace es aplicar el estilo a los botones de incremento o a las partes del incremento del scroll que no cubre la barra deslizable (track-piece).
:decrement
Solamente se puede utilizar con los pseudo-elementos button o track-piece, lo que hace es aplicar el estilo a los botones de decremento o a las partes del decremento del scroll que no cubre la barra deslizable (track-piece).
