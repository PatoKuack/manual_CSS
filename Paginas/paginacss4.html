Layout model

Box model (modelo de caja)

Todos los elementos HTML son rectángulos que con trucos se pueden formar diferentes figuras.
Cada elemento tiene un tamaño que se puede manipular con las propiedades width y height.

Tipos de caja (o elementos)

Bloque

-Cada bloque ocupa todo el espacio horizontal disponible.
-Cada bloque de espacio horizontal crea su propia línea.
-Tienen un ancho (width) y un alto (height).
(ejemplo: h1, p, div, section, ul).

Inline (en línea)

-Ocupa el espacio necesario para su contenido.
-No tienen dimensiones por lo que no se pueden modificar con width o height, se ajustan al contenido
(a excepción de las imágenes porque su tamaño es definido por su resolución propia).
(ejemplo: span, img, a, em, b).

Unidades tamaño (longitud)

Ajusta en pixeles del tamaño original de la letra del elemento (px) - tamaño fijo en cualquier pantalla.
Aumento en % del tamaño original de la letra del elemento (em) - tamaño adaptable en cualquier pantalla.
Aumento en % del tamaño base de la letra de todo el body (rem) - tamaño adaptable en cualquier pantalla.
Aumento en % del tamaño vertical (height) del viewport (vh) - tamaño adaptable en cualquier pantalla.
Aumento en % del tamaño horizontal (width) del viewport (vw) - tamaño adaptable en cualquier pantalla.
Divide en fracciones el área del elemento (fr) - adaptable al tamaño del elemento (se usa generalmente en la propiedad grid), por ejemplo: si se coloca un elemento hijo de 1fr y otro de 3fr, un elemento medirá 25% del tamaño del elemento padre y el otro elemento medirá 75% del elemento padre.
nota1: Un viewport representa la región poligonal (normalmente rectangular) en gráficas de computación que está siendo visualizada en ese instante.
nota2: cuando se requiera un valor de 0 (cero), se coloca SIN una dimensión como por ejemplo 0px o 0%

Display

Permite definir si el elemento en pantalla se define como bloque, línea, ambos o ninguno.

valores de display

  • block
    • Hace que un elemento se comporte como bloque, aunque no lo sea. (ejemplo: span{display: block;})

  • inline
    • Hace que un elemento se comporte como línea, aunque no lo sea. (ejemplo: p{display: inline;})

  • inline-block
    • Hace que un elemento se comporte como inline, pero permitiendo modificar su ancho y largo. (ejemplo: p{display: inline-block;})

  • list-item
    • Hace que un elemento se comporte como el item de una lista no ordenada con sus puntos decorativos a un lado.
  • none
    • Oculta un elemento (se utiliza para mostrarlo al hacer una interacción con la página web u ocultar elementos en dispositivos pequeños). (ejemplo: img{display: none;})
