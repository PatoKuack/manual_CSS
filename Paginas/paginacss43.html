Programación en SASS.

SASS tiene características de un lenguaje de programación por lo que se comporta como uno.

Tipos de datos.

Con la función type-of() se pueden obtener los tipos de datos.

.scss
$varq: 'hello';
$varunq: hello;
.button {
 content: type-of($varq);
 content: type-of($varunq);
}
/* La propiedad content no admite valores sin comillas, pero se representa por esta vez como medio demostrativo.*/
.css


.button {
 content: string;
 content: string;
}
/* La propiedad content no admite valores sin comillas, pero se representa por esta vez como medio demostrativo.*/
Strings
Pueden ser entrecomillados (quote) o no entrecomillados (unquote) y pueden concatenarse (si se suma un número y un string, se concatenan).
.scss
$var: quote(hello);
$const: "yellow";
.button {
 content: $var;
 color: unquote($color);
}
.css


.button {
 content: "hello";
 color: yellow;
}
Números
Pueden ser enteros o decimales.
Colores
Sass reconoce los colores hexadecimales, por nombre, función rgba, rgba o hsl.
Booleans
True o false, se usa para comparaciones.
Lists
Son valores separados por comas y por espacios, como el valor de un borde (1px solid coral) o el de una fuente ('Comic Neue', 'cursive', 'sans-serif').
Maps
Son contenedores de propiedades con sus respectivos valores, a través de map-get se pueden mandar a llamar valores específicos del contenedor.
.scss
$var: (
 color: crimson,
 sizeXH: 1px,
 sizeM: 12px
);

.body{
 color: map-get($var, color);
}
.css






.body{
 color: crimson;
}

Con la función unit() se puede obtener la unidad del valor insertado en su parámetro.

.scss
$varA: 1em;
$varB: 15px;
.button {
 content: 1+(unit($varA));
 content: 1+(unit($varB));
}
.css


.button {
 content: "1em";
 content: "1px";
}

Condicionales.

Las condicionales en SASS se utilizan para escuchar condiciones y ejecutar la que se cumpla o ignorar la que no.

Para las condicionales se utiliza:

  • @if (sí)
  • @else if (sino sí)
  • @else (sino)
  • > (mayor que)
  • < (menor que)
  • == (igual que)
  • # % # (residuo)
  • () and ()
  • () or ()
.scss
$dev: false !default;

@if $dev == true {
 * {
  color: bisque;
 }
} @else if (1==1) and (3>1) {
 * {
  color: orchid;
 }
}
.css


* {
 color: orchid;
}

Condicionando importaciones:

_buttons.scss
@if $buttons {
 .button {
  color: darkseagreen;
 }
}
_colors.scss
@if $colors {
 .color {
  color: lightseagreen;
 }
}
styles.scss
$buttons: true !default;
$colors: true !default;
$colors: false;

@import "buttons";
@import "colors";
styles.css
.button {
 color: darkseagreen;
}

En éste caso se importan los dos archivos partial (los que llevan guion bajo en su nombre), pero no se importa el contenido de los @if que tengan su condicional como false.

Generalmente se utilizan los @if($valor) con un solo valor como condicional para asegurarse de que el valor exista o que sea verdadero, ya que si no existe será igual a falso y la acción de la condicional no se ejecutará.

Ciclos.

Las iteraciones o ciclos se utilizan para repetir muchas veces una acción.

@for

Para crear un ciclo con for, se utiliza la siguiente estructura:
@for $i from (start) through (end){...}

.scss
// $i va de 1 hasta 10.
@for $i from (1) through (10) {
// Se interpola $i.
 .column-#{$i * 10} {
// Se convierte en porcentaje y en decena.
  width: $i * 10%;
 }
}
.css
.column-10{ width:10%; }
.column-20{ width:20%; }
.column-30{ width:30%; }
.column-40{ width:40%; }
.column-50{ width:50%; }
.column-60{ width:60%; }
.column-70{ width:70%; }
.column-80{ width:80%; }
.column-90{ width:90%; }
.column-100{width:100%;}
.scss
@for $i from (1) through (10) {
// si el residuo de $i entre 2 es 0 (si es par).
 @if $i % 2 == 0 {
  .column-#{$i * 10} {
   width: $i * 10%;
  }
 }
}
.css
.column-20{ width:20%; }
.column-40{ width:40%; }
.column-60{ width:60%; }
.column-80{ width:80%; }
.column-100{width:100%;}

@each

El ciclo @each itera sobre las lists y los maps. Para crear un ciclo con each, se utiliza la siguiente estructura:
@each $valores in (list)

.scss
$colors: olive grey gray;

@each $color in $colors {
 .#{$color} {
  border-color: $color;
 }
}
.css
.olive {
 border-color: olive; }
.grey {
 border-color: grey; }
.gray {
 border-color: gray; }
.scss
$colors: (
 primary: olive,
 secondary: grey,
 tertiary: gray
);

@each $key, $value in $colors {
 .button-#{$key} {
  border-color: $value;
 }
}
.css
.button-primary {
 border-color: olive; }
.button-secondary {
 border-color: grey; }
.button-tertiary {
 border-color: gray; }
