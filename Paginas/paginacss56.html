Animaciones SVG.

SVG (Scalable Vector Graphics) es un conjunto de instrucciones XML que el navegador interpreta como una imagen.

Inserciónde SVG.

Existen 3 formas de insertar una imagen SVG:

  • Por medio de la etiqueta img en el documento HTML (imagen HTML).
  • Por medio de la etiqueta background-image en el documento CSS (fondo CSS).
  • Por medio de la inserción del código SVG dentro del documento HTML (inline HTML).

Softwares.

Existen varios softwares para la creación de SVG:

  • Adobe Illustrator (comercial).
  • Affinity Designer (comercial).
  • Inkscape (gratuito).
  • Gravit Designer (versión gratuita y comercial).

Es importante que se verifique el código que se guarda en el SVG ya que puede contener código basura, es decir, exceso de código que no se requiere.

También es importante tomar en cuenta que cuando se crean SVG con texto y ese texto tiene un formato de fuente especial, se debe guardar el SVG creado convirtiendo el texto en contornos, de esta forma el texto se convierte en dibujos dentro del código y el usuario no necesita tener la fuente instalada para visualizar el formato de fuente, aunque esto aumenta las líneas de código; si no se hace esto, el usuario visualizará el texto con un formato de fuente genérica en caso de no tener la fuente instalada.

En la exportación de SVG en Illustrator existen 3 tipos de estilos:

  • CSS interno: Estilos dentro de CSS.
  • Estilo en línea: Estilo dentro de HTML.
  • Atributos de presentación: Similar al HTML antiguo.

Estructura de un SVG.

Se puede definir el tamaño del SVG de la siguiente forma:

<svg width="400" height="300"> </svg>

Existe una etiqueta llamada viewbox que define el sistema de coordenadas interno del SVG.

Las coordenadas del viewbox se calcularán según el ancho y alto renderizados en el gráfico (no es necesario asignar una unidad de media, ya que es un sistema de coordenadas interno).

rect es una etiqueta que crea un rectángulo, se tiene que definir el punto dentro del sistema de coordenadas en donde se comenzará a dibujar (x="#" y="#") y las dimensiones del rectángulo (width="#" height="#").

// Se define el punto inicial (0 0) y el punto final (300 100) obteniendo un sistema de coordenadas de 300 de ancho y 100 de alto.
<svg width="300px" height="100px" viewBox="0 0 300 100">
// Se genera un rectángulo a partir de las coordenadas (10 10) de un ancho de 280px y un alto de 80px.
 <rect x="10" y="10" width="280" height="80" />
</svg>

Con la etiqueta line se puede crear una línea definiendo el punto dentro del sistema de coordenadas en donde se comenzará a dibujar (x1="#" y1="#") y el punto final (x2="#" y2="#"), con stroke se puede definir el color de la línea (stroke="green") y con stroke-width su grosor (stroke-width="#").

Otras propiedades para esta etiqueta son:

stroke-linecap: [none | round | square];
Indica si a la línea se le agregará un medio círculo, un cuadrado o nada en cada extremo.
...
 < line x1="50" y1="20" x2="250" y2="20" stroke="black" stroke-width="10px" stroke-linecap="round" />
...
stroke-dasharray: # #;
Sirve para hacer segmentos. El primer valor es el segmento y el segundo es el espaciado. Se le pueden agregar más de dos valores y el SVG hace un cálculo dinámico entre espaciados y segmentaciones.
...
 < line x1="50" y1="50" x2="250" y2="50" stroke="black" stroke-width="10px" stroke-dasharray="50 15" />
...
stroke-dashoffset: #;
Esta propiedad funciona junto con stroke-dasharray, define desde que parte de la línea comenzará el trazo.
...
 < line x1="50" y1="50" x2="250" y2="50" stroke="black" stroke-width="10px" stroke-dasharray="50 15" stroke-dashoffset="50" />
...

Con la etiqueta polyline se puede generar una figura conformada por varios puntos, se define el punto dentro del sistema de coordenadas en donde se comenzará a dibujar y después se definen los puntos que conforman la figura (points="# #, # #, # #, ...").

Utiliza las mismas propiedades stroke que utiliza la etiqueta line, aunque se pueden utilizar otras como stroke-linejoin: [round | none]; que permite redondear las esquinas en las que se interceptan las líneas de las coordenadas.

Por default se colorea de negro y si no se le agrega una coordenada que cierre la figura, automáticamente se cierra, aunque no se visualizarán las propiedades stroke en esa línea de cierre.

SVG
<svg width="300px" height="100px" viewBox="0 0 200 100">
 <polyline class="saludoSVG" points="20 20, 20 80, ... , 20 20" stroke="currentColor" fill="currentColor" stroke-width="10px" stroke-linejoin="round" />
</svg>
Animación
.saludoSVG{
 stroke-dasharray: 880;
 stroke-dashoffset: 880;
}
.contenedorSaludoSVG:active .saludoSVG{
 animation: draw 2s forwards;
}
@keyframes draw {
 to{
  stroke-dashoffset: 0;
 }
}
Da clic

Con la etiqueta polygon se puede generar una figura igual que polyline, con la diferencia de que, si no se le indica una coordenada de cierre, la figura se cierra automáticamente creando una línea desde la última coordenada a la coordenada inicial (es posible visualizar el diseño de stroke en esta línea de cierre).

Como se vio en el saludo animado, se puede utilizar la propiedad stroke-linejoin=" [none | square | round ] "para que los trazos queden con esquinas redondeadas o cuadradas.

Ya que la etiqueta polyline se genera un relleno por default (como la "o" y la "a" del ejemplo animado), se puede utilizar la propiedad fill="color" para colorear el relleno o eliminar el relleno con el valor transparent.

Con la etiqueta circle se crea un círculo definiendo el punto dentro del sistema de coordenadas en donde se comenzará a dibujar (cx="#" cy="#"), se define el radio (r="#") y su color de relleno (fill="aqua").

<svg width="230px" height="100px" viewBox="0 0 200 100">
 <circle cx="100" cy="50" r="45" fill="aqua" stroke="black" stroke-width="2" />
</svg>

Con la etiqueta ellipse se genera una elipse, es similar a circle con la diferencia de que ellipse requiere dos radios (rx="#" ry="#").

<svg width="230px" height="100px" viewBox="0 0 200 100">
 <ellipse cx="100" cy="50" rx="80" ry="45" fill="aqua" stroke="black" stroke-width="2" />
</svg>

Con la etiqueta text se puede generar un texto definiendo el punto dentro del sistema de coordenadas en donde se comenzará a dibujar (x="#" y="#").

<svg width="230px" height="100px" viewBox="0 0 200 100">
 <text x="10" y="80" font-size="80">
  hola
 </text>
</svg>

Estilos.

Existen dos maneras de dar estilos a un SVG:

  • La primera forma es con estilos en línea como los estilos creados directamente en el documento HTML con la diferencia de que los estilos se colocan dentro de la etiqueta defs en vez de head.
<svg> <defs> <style> .clase { stroke: green; stroke-width: 20; } </style> </defs> </svg>
  • La segunda forma es colocando los estilos directamente dentro de la etiqueta a la que se le quiere aplicar el estilo.
<svg viewbox="0 0 400 300">
 <line x1="10" y1="10" x2="200" y2="150" stroke="green" stroke-width="20" />
</svg>

Es recomendable que los estilos base estén dentro del SVG, no importa cuál de las dos formas se utilice para dar estilos al SVG.
