Propiedades del modelo de caja 2

Border

Línea que esta entre margin y padding, para manipularlo se usa la propiedad y valor border: #px -tipo_de_borde- -color-;
el cual le da al borde un tamaño, tipo de borde y color.
También se puede usar esos valores por separado:
width: 400px;
height: 150px;
background-color: blanchedalmond;
border-width: 4px 8px 16px 32px;
border-style: solid dashed dotted double;
border-color: black blue red green;
En la caja, también se puede hacer un shorthand por cada lado
border-top: #px -tipo_de_borde- -color-
o colocarlo por separado:
border-top-width: #px;
border-top-style: -tipo-;
border-top-color: -color-;
Se pueden utilizar los bordes para hacer triángulos con el valor transparent:
.triangulo{
padding: 0;
border: 50px solid;
border-color: transparent;
border-left-color: black;
}

Border-radius

Propiedad que achata las puntas de las cajas. Con el valor de la propiedad se crea un círculo,
el cual se divide en 4 partes colocando cada parte en las esquinas del borde de la caja.
Cuando se coloca el valor de esta propiedad en pixeles (px), se crea un borde circular ya que se obtiene un único radio definido de los pixeles.
En cambio, cuando se coloca el valor de esta propiedad en porcentaje (%), se crea un borde elíptico
ya que se toma el porcentaje del ancho y del alto y se obtiene un radio mayor y un radio menor.

Circular
elíptica
Al igual que margin y border, se puede utilizar un bordeado por esquina:
border-top-left-radius: #px;
border-top-right-radius: #px;
border-bottom-right-radius: #px;
border-bottom-left-radius: #px;
Y un bordeado especifico utilizando un valor X y un valor en Y.
border-radius: Xpx / Ypx
border-top-left-radius: Xpx / Ypx;
border-top-right-radius: Xpx / Ypx;
border-bottom-right-radius: Xpx / Ypx;
border-bottom-left-radius: Xpx / Ypx;
O se puede usar el showhand:
border-radius: Xpx Xpx Xpx Xpx / Ypx Ypx Ypx Ypx;
Y al igual que margin y border se pueden omitir valores;
si se omite el valor de la cuarta X o Y, se tomará el valor de la segunda X o Y respectivamente,
si se omite el valor de la tercera y cuarta X o Y, se tomara el valor de la primera y segunda X o Y respectivamente.

Box sizing

En una caja se generan el contenido, relleno, borde y margen, todo esto (menos el margen) se suma y aunque se haya definido el width y hight de cierto tamaño,
la caja final será más grande de lo esperado por estas sumas.
Para que la caja final mida lo esperado se utiliza la propiedad box-sizing: border-box;
Un hack recomendado para que se aplique a todo es usando selector universal y pseudoelemento (CSS avanzado) de la siguiente manera:
*, *::before, *::after{box-sizing: border-box}

Box-shadow

Es una sombra de caja con la siguiente estructura:
box-shadow: h-offset v-offset blur spread color inset
h-offset: deslizamiento horizontal (#px o -#px).
v-offset: deslizamiento vertical (#px o -#px).
blur: difuminado (#px).
spread: expansión y contracción (#px o -#px).
color: color, por default toma el color de letra asignado a la caja.
inset: si se coloca, la sombra será interna, si no se coloca, la sombra será externa.

Puede generarse de una forma dinamica con cssgenerator.org
