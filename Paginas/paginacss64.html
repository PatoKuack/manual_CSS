Estilos

En la documentación, en la parte de LAYOUT se encuentran los estilos css de tailwind. La forma de activar tailwind en las etiquetas es con class=""

Las medidas de tailwind son 1 = 4px = .25rem

Se pueden observar todos los estilos desde la documentación a partir del bloque LAYOUT.

Colores.

Tailwind tiene una gran paleta de colores que se puede visualizar en la documentación (Colors), estos colores pueden ser personalizados.

Mediante el playground de tailwind se pueden probar los colores escribiendo etiquetas con sus clases en el lado negro y visualizando el resultado en el lado blanco.

Breackpoints.

El breackpoint es el cambio de layout de la pantalla.

Las medidas dentro de tailwind son:

sm max-width: 640px;
md max-width: 768px;
lg max-width: 1024px;
xl max-width: 1280px;
2xl max-width: 1536px;

Para especificar los estilos para versiones web del proyecto se utiliza el breakpoint especificándolo antes de agregar las clases, ejemplo: lg:w-full lg:h-full

Dark-mode.

Para agregar el modo oscuro en un proyecto, se agrega el siguiente bloque en el archivo tailwind.config.js:

content: [ ... ],
module.exports = {
 darkMode: 'class',
},
 theme: { ... },

Después se agregan las clases que se deseen en el modo oscuro al final de la clase de la siguiente manera: class="bg-white dark:bg-black text-white" ...
