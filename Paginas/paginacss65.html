Plugins

Tailwind cuenta con varios plugins oficiales, pero los más útiles son:

Forms

Permite resetear los estilos predeterminados con los que cuentan los formularios, de esta manera es posible sobrescribir los estilos y trabajar con clases de utilidades dentro de Tailwind.

Typography

Agrega una nueva clase 'prose' que permite dar un mejor formato a estos elementos permitiendo tener el control sobre estilos vanilla con los que cuenta HTML, elementos como headings, paragraphs, listas, entre otros.

Aspect ratio

Aspect ratio es la relación entre el ancho y el alto de la imagen o video. En muchas ocasiones cuando se modifican las dimensiones de elementos o imágenes, se pueden llegar a tener problemas con el aspect ratio, pero con este plugin se resuelve el problema integrando la clase 'aspect' que permite establecer un aspect ratio de manera fija para los elementos.
