Extracción de componentes

La extracción de componentes sirve para no escribir el mismo código en componentes que se utilizarán más de tres veces y poder mantener el principio de utility-first.

La extracción de componentes trabaja de la mano con la directiva @apply.

En la documentación (Reusing styles) se puede saber cómo utilizar la extracción de componentes.

En el archivo src/css/tailwind.css se crean clases con el nombre deseado y con la primera letra en mayúscula, dentro de la clase se introducen todos los estilos de Tailwind que se estarán reutilizando haciendo el uso de la directiva @apply seguido de las clases de Tailwind, ejemplo:

@tailwind base;
@tailwind components;
@tailwind utilities;
.clase-name {
 @apply w-48 h-64 shadow-md rounded-lg
}

Se compila el proyecto y después se podrá utilizar el nombre de la nueva clase en el archivo HTML.

Tailwind no está diseñado para trabajar con muchas clases usando @apply, por lo que se debe identificar y limitar la cantidad de clases que se crearán.
